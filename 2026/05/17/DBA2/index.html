1. java环境变量配置

下载安装包：server-jre-8u221-linux-x64.tar.gz

首先在 /opt 下新建java目录，将压缩包传入 java 下，解压，修改解压后目录名字为 jdk1.8 ，以便修改环境变量

tar -zxvf server-jre-8u221-linux-x64.tar.gz

mv jdk1.8.0_221 jdk1.8

配置环境变量

vi /etc/profile
在文件内容的末尾输入以下内容

使配置生效

**source /etc/profile **
java -version

2. oracle 19c一键安装数据库脚本

1、环境准备(必看)

操作系统：Centos7.9 2009 ×64;

数据库安装包： [LINUX.X64_193000_db_home.zip]

前置脚本：[Oracle19c_preinstall.sh](https://tech.hzlinks.net/xwiki/bin/download/4.通用技术知识库/4.2.数据库/4.2.1.ORACLE/oracle 19c一键安装数据库脚本/WebHome/Oracle19c_preinstall.sh?rev=1.1) --已修改oracle用户弱密码

安装脚本：[ Oracle19c_install.sh](https://tech.hzlinks.net/xwiki/bin/download/4.通用技术知识库/4.2.数据库/4.2.1.ORACLE/oracle 19c一键安装数据库脚本/WebHome/Oracle19c_install.sh?rev=1.2)

安装unzip命令：yum -y install unzip

👉 必读：1、将系统镜像、数据库安装包、两个脚本上传数据库服务器的/opt目录下

2、安装前请检查DNS，将DNS改为学校实际的DNS地址、或者根据业务情况，去除DNS

★建议：

执行一遍脚本里面的安装依赖，目的是测试yum是否可以正常安装Oracle环境需要的依赖包。如果没有报错，则正常执行脚本即可。

yum install -y net-tools bc binutils elfutils-libelf elfutils-libelf-devel fontconfig-devel glibc glibc-devel ksh libaio libaio-devel libXrender libX11 libXau libXi libXtst libgcc libnsl librdmacm libstdc++ libstdc++-devel libxcb libibverbs make smartmontools sysstat

👉支持的国产化操作系统

关于支持的操作系统：龙蜥8.9：

如果用的系统镜像为：AnolisOS-8.9-x86_64-minimal.iso 可以直接安装

如果用的系统镜像为：AnolisOS-8.9-x86_64-dvd.iso 需要安装依赖：yum install -y glibc-devel libnsl

1️⃣ 修改建库相关配置 Oracle19c_install.sh的58至72行

如果不想使用Oracle19c的容器模式，可以把第58行的 echo "createAsContainerDatabase=true"中的true 改成 false

然后不用关注下面于pdb相关的内容，这样安装出来的Oracle与11g版本的一样是单实例。

Oracle原本默认总内存为2G，现修改为8G

为了兼容非centos系统安装，在执行runInstaller脚本前修改一个安装配置文件如下，安装脚本有更新，请重新下载安装脚本

echo ‘CV_ASSUME_DISTID=OL7’ >>/u01/app/oracle/product/19.3.0/dbhome_1/cv/admin/cvu_config
